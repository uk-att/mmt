# My App

A simple web UI project with a form to save name and address.

## How to Run

### Backend
1. Open a terminal in `server` folder
2. Run `npm install`
3. Run `npm start` (server runs on http://localhost:3001)

### Frontend
1. Open a terminal in `client` folder
2. Run `npm install`
3. Run `npm run dev` (frontend runs on http://localhost:5173)

## Usage
- Fill in the form and click Save. Data is sent to the backend and stored in memory.
